# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-08

### Added

- Initial release
- Spherical and ellipsoid deformation support
- Three deformation modes: Radial, Normal, Surface
- NDMF integration with real-time preview
- Scene view editing handles (Move, Rotate, Scale)
- Custom Inspector with sphere selection
